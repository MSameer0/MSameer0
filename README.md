I'm currently studying **Computer Science at FAST NUCES** and building games, tools, and software projects both independently and with friends.

My programming journey started before university through **Minecraft mod development**, where I created custom mods and worked with clients through freelance commissions. Since then, I've expanded into game development, desktop applications, Android development, and collaborative software projects.
